# ⚛️ React Hooks Examples

This project demonstrates **all commonly used React Hooks** with simple and clear examples.  
It helps developers understand **how and when to use React Hooks in real-world applications**.

---

## 🚀 Hooks Covered

The following hooks are included in this project:

- useState
- useEffect
- useContext
- useRef
- useReducer
- useMemo
- useCallback
- useLayoutEffect
- useImperativeHandle
- useId
- useTransition
- useDeferredValue
- useDebugValue

Each hook has its **own example component** for better understanding.
